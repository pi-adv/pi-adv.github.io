Tools for Out of Competition

Welcome to the helpful tool page for tools that should be used out of competition.

To clarify these tools can be used in competition but they are courses or more time-consuming tools than other tool on the Tools Inside Competition page.

Section 1 - Courses

The first great resource that you can find to enhance your skills is courses. More specifically, courses provided by CyberPatriot themselves.

Going to this CyberPatriot website you can find many different, official courses that are provided from cyberpatriot.

In these courses the length will span from 2 hours to even a 70 hour course. This is fantastic since if you are new or experienced there should be something here for you to try out.

Some Recommeneded Courses:

  • Introduction to Cybersecurity (6 Hours)
  • Operating System Basics (12 Hours)
  • Networking Basic (22 Hours)
